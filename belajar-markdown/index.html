Belajar Markdown untuk Pemula

Pengenalan

Apa sih Markdown itu?

Markdown adalah bahasa markah ringan dengan sintaksis pemformatan teks polos. Bahasa markah ini dirancang agar bisa dikonversi menjadi HTML dan banyak format lainnya menggunakan alat yang memiliki nama yang serupa. Markdown sering digunakan untuk memformat file README, untuk menulis pesan di forum diskusi online, dan untuk membuat teks kaya menggunakan penyunting teks polos. Karena deskripsi awal tentang Markdown mengandung ambiguitas dan pertanyaan yang tidak terjawab, banyak implementasi dan perluasan Markdown muncul selama bertahun-tahun untuk menjawab permasalahan ini.

Sejarah Markdown

John Gruber menciptakan bahasa markdown pada tahun 2004 bekerja sama dengan Aaron Swartz yang bekerja untuk sintaksisnya, dengan tujuan untuk memungkinkan orang “menulis menggunakan format teks polos yang mudah dibaca, mudah untuk tulis, dan secara opsional mengubahnya menjadi XHTML (atau HTML) yang valid secara struktural”.

Tujuan dari desain utamanya adalah keterbacaan â€“ yaitu bahwa bahasa itu dapat dibaca seperti apa adanya, tanpa terlihat seperti ditandai dengan tag atau instruksi pemformatan, tidak seperti teks yang terformat dengan bahasa markah, seperti Rich Text Format (RTF) atau HTML, yang memiliki tag dan instruksi pemformatan yang mencolok. Untuk tujuan ini, inspirasi utamanya adalah konvensi yang ada untuk menandai teks biasa dalam email, meskipun juga mengambil ide dari bahasa markah sebelumnya, terutama setext, Textile, dan reStructuredText.

Gruber menulis sebuah skrip Perl, Markdown.pl, yang mengkonversi teks yang sudah termarkah menjadi XHTML atau HTML yang terformat dengan baik, mengganti tanda kurung sudut kiri ‘<’ dan ampersan ‘&’ dengan referensi entitas yang sesuai. Ini bisa digunakan sebagai script standalone, sebagai plugin untuk Blosxom atau Movable Type, atau sebagai penyaring teks untuk BBEdit.

Markdown kemudian di implementasikan kembali sebagai modul Perl yang tersedia di CPAN (Text::Markdown), dan dalam berbagai bahasa pemrograman lainnya. Markdown di distribusikan dibawah lisensi bergaya BSD dan disertakan secara baku, atau tersedia sebagai plugin pada beberapa sistem manajemen konten (CMS).

Situs-situs seperti GitHub, Bitbucket, Reddit, Diaspora, Stack Exchange, OpenStreetMap, dan SourceForge menggunakan varian Markdown untuk memfasilitasi diskusi antar pengguna.

Kenapa menggunakan Markdown?

Kamu mungkin berpikir kenapa orang-orang menggunakan Markdown daripada HTML. Berikut ini beberapa alasan kenapa orang-orang lebih memilih menggunakan Markdown daripada menggunakan HTML.

  • Markdown bisa digunakan untuk segalanya. Orang-orang menggunakan Markdown untuk membuat website, dokumen, catatan, buku, presentasi, pesan email, dan dokumentasi teknik. Jika kamu tidak suka dengan aplikasi pengolah Markdown yang sedang kamu gunakan, kamu bisa mengimpor file Markdown itu ke aplikasi pengolah Markdown lainnya tanpa menulisnya ulang ðŸ“‹.

  • Markdown itu tidak bergantung pada platform. Kamu bisa membuat teks berformat Markdown di perangkat apapun dan sistem operasi apapun. Markdown tingkat Dasar

Sintaks Markdown ini adalah sintaks Markdown yang didukung semua aplikasi pengolah Markdown.

Heading

Kamu bisa menggunakan Heading di Markdown layaknya di HTML. Disini saya sertakan tabel perbandingan Sintaks Markdown dan Sintaks HTML, semakin besar nomor-nya berarti semakin kecil ukuran Heading nya

No Sintaks Markdown Sintaks HTML
1 # Heading 1 <h1>Heading 1</h1>
2 ## Heading 2 <h2>Heading 2</h2>
3 ### Heading 3 <h3>Heading 3</h3>
4 #### Heading 4 <h4>Heading 4</h4>
5 ##### Heading 5 <h5>Heading 5</h5>
6 ###### Heading 6 <h6>Heading 6</h6>

Jika kita perhatikan, jumlah tandah Hashtag # berjumlah sama dengan nomor Heading nya yang berarti Semakin banyak tanda # maka semakin kecil ukuran Heading nya.

Bold

Markdown juga mendukung penulisan Huruf Tebal / Bold. Cara penulisannya yaitu dengan mengapit huruf yang akan kita tebalkan dengan tanda **. Contoh nya seperti **Tebal** menghasilkan Tebal. Berikut ini perbandingan Sintaks Markdown dan Sintaks HTML untuk menebalkan huruf.

Sintaks Markdown Sintaks HTML Hasil
**Huruf Tebal** <b>Huruf Tebal</b> Huruf Tebal

Italic

Tidak hanya Markdown dan Heading. Markdown juga mendukung penulisan Huruf Miring / Italic. Cara penulisannya hampir sama dengan Bold hanya saja Italic menggunakan tanda _ untuk mengapit huruf. Contoh nya seperti _Miring_ menghasilkan Miring. Berikut ini perbandingan Sintaks Markdown dan Sintaks HTML untuk memiringkan huruf.

Sintaks Markdown Sintaks HTML Hasil
_Huruf Miring_ <i>Huruf Miring</i> Huruf Miring

Ordered List

Markdown juga bisa digunakan untuk membuat Pengurutan Terstruktur / Ordered List. Cara menggunakannga yaitu dengan menggunakan nomor didepan item yang akan kita urutkan. Contoh Kode sebagai berikut (Ingat! Nomor didepan item bukan nomor baris, melainkan sintaks yang digunakan untuk membuat Pengurutan Terstruktur).

1. Item ke 1
2. Item ke 2
3. Item ke 3

Kode diatas akan menghasilkan tampilan berikut ini.

  1. Item ke 1
  2. Item ke 2
  3. Item ke 3

Unordered List

Selain Ordered List, Markdown juga mendukung Pengurutan Tidak Terstruktur / Unordered List. Cara menggunakannya juga cukup mudah, hanya dengan menambahkan tanda - didepan item yang akan kita urutkan. Contoh Kode sebagai berikut.

- Item ke 1
- Item ke 2
- Item ke 3

Kode diatas akan menghasilkan tampilan berikut ini.

  • Item ke 1
  • Item ke 2
  • Item ke 3

Code

Markdown juga bisa digunakan untuk menulis kode loh. Hanya dengan menambahkan tanda ` untuk mengapit kode kita (Kode yang digunakan sebaiknya kode sebaris / One line code). Contoh Kode sebagai berikut.

`for i in range(0, 10)`

Kode diatas akan menghasilkan tampilan berikut ini.

for i in range(0, 10)

Horizontal Line

Kamu juga bisa membuat Garis Horizontal / Horizontal Line dengan Markdown. Caranya yaitu dengan menambahkan --- di kode kita. Contoh Kode sebagai berikut.

<!-- Kode sebelum garis -->
---
<!-- Kode setelah garis -->

Kode diatas akan menghasilkan tampilan berikut ini.

Link

Link / Hyperlink juga bisa dibuat dengan Markdown. Cara penggunaannya juga cukup mudah, hanya dengan mengetikkan [Teks yang ditampilkan](https://link-yang-akan-dituju). Contoh Kode sebagai berikut.

[Link Google](https://google.com)

Kode diatas akan menghasilkan tampilan berikut ini.

Link Google

Image

Image / Gambar juga bisa ditampilkan dengan menggunakan Markdown. Sintaks nya hampir sama dengan Link, hanya saja ditambahkan ! didepan sintaks nya. Contoh Kode sebagai berikut.

![Logo Hugo](https://d33wubrfki0l68.cloudfront.net/c38c7334cc3f23585738e40334284fddcaf03d5e/2e17c/images/hugo-logo-wide.svg)

Kode diatas akan menghasilkan tampilan berikut ini.
